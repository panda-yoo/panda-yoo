## What I'm working on

**LBM simulation of droplet dynamics**
Phase-field Lattice Boltzmann simulation using Palabos (C++).
Validated against Poiseuille flow; extending to reaction-diffusion 
coupling to look at how chemical waves drive interface deformation.

**Droplet detection — image analysis**
Labeled ~700 images in CVAT and trained a YOLO-based model to 
detect droplets. The goal was to connect what the model sees to 
a simpler macroscopic physical model — trying to get data-driven 
and theory-driven approaches to talk to each other.

**Quantum ML (exploring)**
Took a course on quantum computing and worked on a quantum SVM 
using Qiskit — experimenting with how different kernel methods 
and feature maps affect classification. Still early, but it connects 
to the same thing that drew me to the Ising model: a clean set of 
rules producing behaviour that's hard to anticipate classically.
